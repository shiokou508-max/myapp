# TOEFL 3800 Vocabulary Quiz App

A modern, glassmorphism-styled web application for learning TOEFL vocabulary. This app provides quizzes and word lists for 3800 essential words, categorized by rank.

## Features

-   **4 Ranks**: Vocabulary is divided into 4 difficulty levels (Rank 1-4).
-   **Quiz Mode**: Test your knowledge with multiple-choice questions.
-   **Memory Tracking**: The app tracks your "memory level" for each word (Monoxer-style dots).
-   **Review Mode**: Review incorrect answers after each session.
-   **Word List**: Browse all words in each rank.
-   **Mobile Friendly**: Fully responsive design for smartphones and desktops.
-   **Data Persistence**: Progress is saved automatically in your browser's local storage.

## Installation / Usage

1.  Download this repository.
2.  Open `index.html` in any modern web browser (Chrome, Edge, Safari, FireFox).
3.  Start learning!

## Technologies

-   HTML5
-   CSS3 (Variables, Flexbox, Grid, Glassmorphism)
-   Vanilla JavaScript (ES6+)

## Project Structure

-   `index.html`: The main application interface.
-   `style.css`: All styling rules.
-   `script.js`: Application logic (quiz flow, state management).
-   `vocab.js`: The vocabulary dataset (JSON format).
